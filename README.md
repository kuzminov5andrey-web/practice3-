# Ассемблер и интерпретатор УВМ (вариант 13)

Проект реализует ассемблер и интерпретатор учебной виртуальной машины
(УВМ) по заданной спецификации.

------------------------------------------------------------------------

## Формат программы (JSON)

``` json
{
  "program": [
    {
      "opcode": "НАЗВАНИЕ",
      "fields": {
        "A": число,
        "B": число,
        "C": число,
        "D": число
      }
    }
  ]
}
```

-   **A** --- код операции УВМ\
-   **B, C, D** --- аргументы команды

------------------------------------------------------------------------

## Поддерживаемые команды

-   **LOAD_CONST** (A = 100)
-   **READ_MEM** (A = 110)
-   **WRITE_MEM** (A = 109)
-   **BITREVERSE** (A = 77)

------------------------------------------------------------------------

## Этап 1 --- Ассемблер

Ассемблер читает JSON-программу и формирует бинарное или текстовое
представление команд.\
В режиме тестирования выводит внутреннее представление команд (поля и
значения).

### Запуск

``` bash
python asm.py input.json output.bin --test --text
```

------------------------------------------------------------------------

## Этап 2 --- Интерпретатор и память

Интерпретатор выполняет команды УВМ и сохраняет дамп памяти в формате
JSON.

### Запуск

``` bash
python interpreter.py array_copy.json memory_dump.json 90 210
```

------------------------------------------------------------------------

## Этап 3 --- BITREVERSE

Реализована операция `bitreverse()` и тестовая программа для проверки её
корректности.

### Запуск

``` bash
python interpreter.py bitreverse_test.json bitreverse_dump.json 495 510
```

Результат работы сохраняется в файле `bitreverse_dump.json`.
